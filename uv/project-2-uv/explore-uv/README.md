# What is UV?

UV is a super-fast Python package manager (faster alternative to pip/poetry).

## Common UV Commands

| Action            | Command                                |
|-------------------|----------------------------------------|
| Create new project| `uv init --package myproject`          |
| Add a package     | `uv add crewai`                        |
| Install UV        | `curl -Ls https://astral.sh/uv/install.sh` |
